## Описание
Telegram-бот для отслеживания текущей цены для некоторой валюты, криптовалюты, изменения цены на нефть и драгметаллов с логированием в ElasticSearch

## Технологии использованные в проекте

- Java 17
- Spring Framework
- Telegrambots client
- Redis
- Lombok
- ElasticSearch
- Logstash
- Kibana
- Maven
